<span data-ttu-id="1f262-106">Componentele incluse sunt componente care fac obiectul:</span><span class="sxs-lookup"><span data-stu-id="1f262-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="1f262-109">Setările frecvenței facturilor definite pe linia de contract bazată pe proiect</span><span class="sxs-lookup"><span data-stu-id="1f262-109">Invoice frequency settings defined on the project-based contract line</span></span>

<span data-ttu-id="1f262-110">Un subset al componentelor incluse poate fi marcat ca taxabil folosind câmpul **Tip de facturare**.</span><span class="sxs-lookup"><span data-stu-id="1f262-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="1f262-111">Câmpul **Tip de facturare** este un set de opțiuni care permite următoarele valori în contextul unei linii contractuale:</span><span class="sxs-lookup"><span data-stu-id="1f262-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="1f262-113">Netaxabil</span><span class="sxs-lookup"><span data-stu-id="1f262-113">Non-chargeable</span></span>

<span data-ttu-id="1f262-114">Componentele taxabile pot fi definite pe sarcini, roluri și categorii de tranzacții.</span><span class="sxs-lookup"><span data-stu-id="1f262-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>
