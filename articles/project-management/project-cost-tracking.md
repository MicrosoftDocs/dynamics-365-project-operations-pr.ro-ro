## <a name="labor-cost-tracking-view"></a><span data-ttu-id="71a40-108">Vizualizare urmărire cost forță de muncă</span><span class="sxs-lookup"><span data-stu-id="71a40-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="71a40-109">Pe pagina **Proiecte**, pe fila **Urmărire**, puteți selecta **Urmărire** > **Cost** pentru a deschide vizualizarea **Urmărirea costurilor** și vedeți progresul cheltuielilor de muncă pentru fiecare sarcină din planul de proiect.</span><span class="sxs-lookup"><span data-stu-id="71a40-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="71a40-110">Această viziune compară, de asemenea, costul real al forței de muncă cheltuit pentru o sarcină cu costul forței de muncă planificat al sarcinii.</span><span class="sxs-lookup"><span data-stu-id="71a40-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="71a40-111">Project Operations utilizează următoarele formule pentru a calcula valorile costului forței de muncă:</span><span class="sxs-lookup"><span data-stu-id="71a40-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="71a40-112">**Cost planificat**: Costul estimat al tuturor atribuirilor de resurse pentru fiecare sarcină de nod frunză</span><span class="sxs-lookup"><span data-stu-id="71a40-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>

- <span data-ttu-id="71a40-114">**Procentul de consum al costurilor**: Costul real ÷ Estimarea costului la final</span><span class="sxs-lookup"><span data-stu-id="71a40-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="71a40-115">**Costul rămas**: estimarea de cost la final - costul real</span><span class="sxs-lookup"><span data-stu-id="71a40-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="71a40-116">**Costul la final**: costul rămas + costul real</span><span class="sxs-lookup"><span data-stu-id="71a40-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="71a40-117">**Varianța de cost**: cost planificat - estimarea de cost la final</span><span class="sxs-lookup"><span data-stu-id="71a40-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="71a40-118">Fiecare activitate afișează o proiecție a variației de cost la activitate.</span><span class="sxs-lookup"><span data-stu-id="71a40-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="71a40-119">Dacă estimarea completă a costurilor este mai mare decât costul planificat, sarcina este proiectată să depășească bugetul.</span><span class="sxs-lookup"><span data-stu-id="71a40-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="71a40-120">Dacă estimarea completă a costurilor este mai mică decât costul planificat, activitatea este proiectată pentru a se finaliza sub buget.</span><span class="sxs-lookup"><span data-stu-id="71a40-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

> - <span data-ttu-id="71a40-125">Fila **Estimări** calculează costul estimat utilizând aceeași monedă a ratei de cost din lista de prețuri.</span><span class="sxs-lookup"><span data-stu-id="71a40-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="71a40-126">Apoi, costul estimat în moneda listei de prețuri este convertit în costul estimat în moneda costului proiectului.</span><span class="sxs-lookup"><span data-stu-id="71a40-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="71a40-127">Costul estimat în moneda proiectului este afișat rotunjit la 2 zecimale.</span><span class="sxs-lookup"><span data-stu-id="71a40-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="71a40-128">În niciun moment al acestei conversii nu se aplică precizia monedei.</span><span class="sxs-lookup"><span data-stu-id="71a40-128">At no point during this conversion is currency precision applied.</span></span>
