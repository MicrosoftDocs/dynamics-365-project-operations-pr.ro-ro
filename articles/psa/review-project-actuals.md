# <a name="review-project-actuals-project-service"></a>Revizuiți valorile actuale ale proiectului (Project Service)

[!INCLUDE[cc-applies-to-psa-app-1x-2x](../includes/cc-applies-to-psa-app-1x-2x.md)]
