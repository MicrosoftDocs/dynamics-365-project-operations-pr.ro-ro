# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="20e1c-103">Configurarea și utilizarea plăților de la furnizor cu plată la plată</span><span class="sxs-lookup"><span data-stu-id="20e1c-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="20e1c-104">Când aprobați un furnizor să lucreze ca subcontractant, este posibil să doriți să rețineți plata către furnizor până când clientul dvs. vă plătește pentru proiect.</span><span class="sxs-lookup"><span data-stu-id="20e1c-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="20e1c-105">Pentru a susține acest scenariu, puteți configura termeni de plată la plată (PWP) atunci când configurați comanda de cumpărare (PO) cu furnizorul.</span><span class="sxs-lookup"><span data-stu-id="20e1c-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="20e1c-106">De exemplu, atunci când un client plătește o sumă pe o factură de proiect, puteți elibera o parte sau o parte din suma facturii furnizorului.</span><span class="sxs-lookup"><span data-stu-id="20e1c-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="20e1c-107">Trebuie doar să configurați termeni PWP care specifică faptul că furnizorul va fi plătit după ce veți primi un procent din plata aferentă de la client.</span><span class="sxs-lookup"><span data-stu-id="20e1c-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="20e1c-108">Dacă primiți o plată parțială de la un client, puteți elibera manual unele dintre facturile aferente furnizorului pentru plată.</span><span class="sxs-lookup"><span data-stu-id="20e1c-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="20e1c-109">Următorul exemplu prezintă procesul când sunt folosiți termenii PWP.</span><span class="sxs-lookup"><span data-stu-id="20e1c-109">The following example outlines the process when PWP terms are used.</span></span>
